* Le tutoriel complet de Godot, fait par Arcadia: https://godot-cours-arcadia-tower-protector-tutoriel.readthedocs.io/fr/iceknith-1.1/

* Le tutoriel officiel de godot pour faire un jeu 3D: https://docs.godotengine.org/en/stable/getting_started/first_3d_game/index.html
* Comment faire des animations dans Godot: https://www.kodeco.com/44643336-introduction-to-godot-s-animationplayer
* Comment faire des shaders dans Godot: https://docs.godotengine.org/en/stable/tutorials/shaders/your_first_shader/index.html
